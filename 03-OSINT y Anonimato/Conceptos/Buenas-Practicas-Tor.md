## Mejores Prácticas (QUÉ HACER)

### Configuración inicial

- Descargar solo de fuentes oficiales (`torproject.org`)
- Verificar firmas digitales
- Actualizar regularmente
- Usar en sistema limpio (evitar malware preexistente)

### Durante la navegación

- Mantener JavaScript desactivado
- Usar siempre HTTPS
- Evitar plugins (Flash, Java, ActiveX)
- Cerrar Tor completamente al terminar sesiones

### Seguridad adicional

- Usar VPN + Tor
- Activar cortafuegos
- Usar sistemas operativos seguros (Tails, Whonix)
- Verificar circuitos y cambiar identidad regularmente

---

## Prácticas Prohibidas (QUÉ NO HACER)

### Configuración incorrecta

- No modificar configuraciones predeterminadas
- No instalar extensiones
- No usar versiones pirata
- No compartir perfil

### Errores de navegación

- No iniciar sesión en cuentas personales
- No descargar archivos (riesgo de malware)
- No usar torrents
- No habilitar JavaScript

### Errores de seguridad

- No usar Tor para actividades ilegales
- No confiar ciegamente, verificar siempre
- No usar HTTP (solo HTTPS)
- No reutilizar contraseñas

---

## Amenazas Comunes

### Ataques de correlación

- Análisis de tráfico (entrada y salida)
- Timing attacks (patrones temporales)
- Mitigación: usar múltiples identidades

### Compromiso de nodos

- Nodos maliciosos controlados por atacantes
- Nodos gubernamentales con monitoreo estatal
- Mitigación: cambiar circuitos regularmente
